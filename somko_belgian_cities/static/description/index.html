Advantages

This module loads all the Belgian cities and provinces in Odoo. This way you don't have to type in the name of the city and the zip code.

You only have to select the name of the city from a list and the zip code and province will be automatically filled in.

Configuration

Install the 'somko_belgian_cities' module.

Open the details of the country "Belgium" ("Contacts" > "Configuration" > "Countries") and mark the parameter "Enforce cities".
